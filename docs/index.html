About

All Mine is a puzzle game based on old 8-bit games like Boulder Dash, Repton, and The Pit. You must explore each level, digging through the earth to collect all the diamonds, before progressing to the next.

Beware of falling boulders and lizards hatching from monstrous yellow eggs.

Some diamonds are locked away in safes which need a key to unlock. And keep an eye out for the wandering spirits who must be guided into cages in return for a diamond …

Controls

  • Z - move left
  • X - move right
  • K - move up
  • M - move down
  • P - pause (and see a map)

Choose a level …

Random levels are created using a seed to generate random numbers. The same seed produces the same level. Random levels are not guaranteed to be possible — try and get the highest score you can!
